Installation¶

GeoPandas is pre-alpha software. Please install the latest source from GitHub and use the setup script:

python setup.py install

Dependencies¶

  • numpy
  • pandas
  • shapely
  • fiona
  • descartes
  • matplotlib

Testing¶

To run the current set of tests from the source directory, run:

nosetests -v

from a command line.
